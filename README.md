This project is to use Hermes Agent to remotely control NL5 to run circuit simulation.   
Step 1: set up the interface between Hermes Agent and NL5  
Step 2: create example python file of HTTP command to drive NL5  
Step 3: set up NL5 simulation test bench   
Step 4: set up design variables and design target  
Step 5: define Hermes task to accomplish the design target
